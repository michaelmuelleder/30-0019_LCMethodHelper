Step-by-step guide to setup the method!

 

There are several SOP methods for the analysis of free amino acids, nucleosides and more general methods for polar metabolites either by IP-RP HPCL or by HILIC.

At the moment we aim not to use ion pairs and try to do all analysis either by RP or HILIC.

 

Although the previous person should have cleaned, do what is necessary and take responsibility for your own analysis.

Preparation

  • Prepare worklist This should be a *.csv File with 2 columns:

Sample Name

Sample Position

WT-100H2O2-30sec-r1

P1-A1

MUT-100H2O2-30sec-r1

P1-A2

...

...

 

All sample names have to be unique! No underscores are allowed in the sample name!

  • Start LC-MS/MS-Helper app The app will calculate the required volumes of buffers. Alternatively you can calculate the volumes for a given number of samples. Always state an additional volume to allow for purging, repetition of samples and other problems.
    • Select the acquisition method and load the worklist.
    • Go to Worklist4Masshunter
      • add the number of standards for external calibration
      • select the version of the method you want to use. Sometimes there is the option to include the transitions for 13C-labelled internal standards.
      • Specify the location where the data will be saved
      • Download the worklist file
  • Start Masshunter Acquisition software
  • Import Worklist
    • Navigate to the worklist panel and clear previous samples
    • Right click on top left corner and import the previously generated worklist *.csv file. Accept the creation of a new folder. If not already in the table, add Dilution and Injection Volume. Set injection volume to "As method" and the Dilution to 1 of the first sample. Right click the respective box and "Fill Column".
    • Right click on the top left corner. Here the details for the standby
  • Select method of choice
  • Autosampler
    • Check if there are still samples from previous person, transfer to fridge in common area and inform responsible person.
    • Set the format of the plates, e.g. to 54 Vials or 384 MTP
    • Switch on the cooling unit
    • Check collected condensed water below the LC, empty if necessary.
    • Same for waste bottle. Add relevant stickers and labels and install a new bottle.

LC

  • Remove column if still present and connect lines with the 0-volume connector
  • Remove solvent bottles Take out the solvent lines with filter, rinse with appropriate solvent of spray bottle and place in beaker with clean solvent
    • e.g. change from 0.1% acetic acid to 100 mM ammonium carbonate: rinse with water and place in beaker with water
  • Purge lines usually 3 min with 5 ml/min per channel should be sufficient. Include Priming if concerned about bubbles in lines.
  • Flush capillaries
    • - Set the low pressure limit to 0 in the method tab and press "apply" or enter (Do not save the method!).
    • - Set the flow/min to 1 ml/min and choose appropriate channel to remove buffer mix present in capillaries (e.g. water). Increase flow as deemed necessary. Monitor the pressure, it should be stable.
  • Install bottles with your solvents. Set the solvent counter and volumes appropriately in the Binary Pump panel.
  • Set solvent volumes
  • Purge lines as before
  • Select method of choice and stop flow.
  • Flush capillaries as before
  • Check in-line filter Change solvent ratio to 80-100% organic solvents. If the pressure is greater 50 bar, then exchange the in-line filter.
  • Install column

MS

  • Assure that the source has cooled down or is cool enough the clean
  • wipe the spray shield with water and with 50% 2-propanol
  • remove the spray shield and wipe the cap of the dielectric capillary
  • reinstall and close the source

Nebulizer problems

If signals are poor or precipitation on the nebulizer needle is suspected

  • Put MS into stand-by
  • Turn gas flows to minimum
  • remove the nebulizer and gas lines and capillaries
  • inspect with the appropriate tools
  • place cut pipette tip over nebulizer so the needle that is protruding slightly can't get damaged
  • wash in water and 50% 2-propanol
  • if not sufficient, assist the cleaning with sonication
    • CAREFUL with the needle - can get damaged very easily! Do not use any abrasive paper!
  • Install the nebulizer and reload you method of choice

Acquisition

  • Switch on MS/MS and column compartment and wait until ready
  • Place your samples in the autosampler
  • Load the first method of the worklist. This should either be a equilibration method, or the acquisition method you wanted.

Final check

  • Waste bottle
  • Solvent counter - stop when running out of solvent min. 50 ml
  • Standy-by after worklist is complete
  • any leaks? too low or too high pressure?
  • line connected to MS?

Start

  • Start worklist
  • Wait for equilibration and for first standards to run

Inspect first standards and adjust method

  • Open qualitative Masshunter analysis software
  • Open the standards file
  • Run Find by MRM
  • Inspect if compounds elute at expected retention times and windows are broad enough
  • If necessary, adjust the acquisition method file
